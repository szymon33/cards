# Simple React Deck of Cards application

## Introduction

Technology stack:

* React
* Jest
* Babel
* Webpack
* ESLint
* pure.css (not Bootstrap this time)


## Description

## Installation

* Clone the repo
* Run `npm install`
* Run `npm start`

At this point you should see the app on

```
http://localhost:3000/
```

## Automatic tests

```
npm test
```
